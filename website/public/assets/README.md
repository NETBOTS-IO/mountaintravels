# Mountain Travels Pakistan - Image Structure

This folder contains all the images used throughout the Mountain Travels Pakistan website.

## Folder Structure

### /assets/logo
- **logo.png**: Main company logo used in header and footer

### /assets/home
- **hero-1.jpg**: Mountain landscape with snow-capped peaks
- **hero-2.jpg**: Cultural scene with traditional architecture
- **hero-3.jpg**: Adventure scene with trekkers

### /assets/about/team
- **image1.jpg**: Team member 1
- **image2.jpg**: Team member 2
- **image3.jpg**: Team member 3
- **image4.jpg**: Team member 4

### /assets/tours
- **tour-1.jpg**: K2 Base Camp
- **tour-2.jpg**: Hunza Valley
- **tour-3.jpg**: Fairy Meadows
- **tour-4.jpg**: Deosai Plains

### /assets/contact
- **office.jpg**: Office building exterior

## Tour-Specific Images

For tour-specific images, use the following structure:
- `/assets/tours/[tour-id]/image1.jpg`
- `/assets/tours/[tour-id]/image2.jpg`

Example:
- `/assets/tours/k2-base-camp/image1.jpg`
- `/assets/tours/fairy-meadows/image1.jpg`

## Blog Images

For blog post images, use:
- `/assets/blogs/[blog-id]/image1.jpg`

## Image Guidelines

1. All images should be optimized for web (compressed)
2. Recommended sizes:
   - Hero images: 1920x1080px
   - Tour thumbnails: 800x600px
   - Team photos: 400x400px (square)
   - Logo: 200px height (with transparent background)
3. Use JPG for photos, PNG for graphics with transparency
